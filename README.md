## 🕹️ About RiftAscent

RiftAscent is a canvas-based action game. You jump into a fast-paced world with no limits. The game features prestige cycles, upgrades, co-op play, and new sounds created each time you play. RiftAscent soon will be available on iOS and Android too.

This game runs in your Windows computer's browser using HTML5 and JavaScript. You do not need any special programs or coding skills to use it. The game plays smoothly with simple controls and clear visuals.

Topics include action, coop, canvas graphics, procedural audio, and roguelite style gameplay.

## 🎮 How to Play RiftAscent

Once you start RiftAscent, the game loads in a window or full screen. Controls are simple:

- Use arrow keys or WASD to move your character.  
- Press spacebar to jump or interact.  
- Use mouse clicks for aiming or attacks in co-op mode.  
- Menus appear on screen for upgrades and prestige options.

Your goal is to climb higher in the game's world. Each run lets you earn upgrades. When ready, use the prestige cycle to reset progress and become stronger for the next run.

Co-op mode lets you team up with friends. All players join one game session and work together.
